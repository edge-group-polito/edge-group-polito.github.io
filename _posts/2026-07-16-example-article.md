Delete this file once there is a real article — it exists to show the format.

Copy it to `_posts/YYYY-MM-DD-some-slug.md`, keep the front matter above, write Markdown
below. It appears on `/articles/` and on the landing page automatically. The date in the
filename must match the `date:` field, and a future date will not be published.

The first paragraph becomes the excerpt shown in the listings, so make it say something.

## Headings, code and images work

```c
int main(void) { return 0; }
```

Images go in `assets/img/` and are referenced as `![alt](/assets/img/file.png)`.
